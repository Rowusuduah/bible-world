## v0.1 FEATURE SCOPE (What's In, What's Out)

### IN (v0.1)
- Git diff prompt change detection
- YAML config format
- LLM-as-judge scorer (Anthropic + OpenAI via LiteLLM)
- Semantic similarity scorer (sentence-transformers, offline)
- Exact match scorer
- Judge calibration with agreement rate + Spearman r + bias
- SQLite trace ledger
- `prompt-lock check` CI gate (hard + soft + regression modes)
- `prompt-lock calibrate` command
- `prompt-lock baseline update` command
- `prompt-lock traces show` query
- GitHub Actions integration
- PyPI package

### OUT (v0.1 — future versions)
- Web dashboard (v0.2)
- Postgres trace backend (v0.2)
- HTML report generation (v0.2)
- Multi-model A/B comparison (v0.2)
- Slack/PagerDuty alerts (v0.3)
- Team mode (shared baselines, approval workflows) (v0.3)
- RAGAS integration (v0.2)
- Cost tracking per eval run (v0.2)
- Automatic calibration dataset generation from production logs (v0.4)
